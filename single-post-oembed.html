Mastodon Single Post Embed

Using the Default Mastodon Embed Code

You can quickly grab code for any Mastodon post by choosing the "Get embed code" option from the [...] menu on the Mastodon web interface when the post is loaded. This is the simplest method for single posts. It will likely give you the same blockquote code shown below.

Using the oEmbed API

Another method is to programmatically get the embed code directly via the oEmbed API. For example, the embed below (from macaw.social, which is where my own account lives) could be retrieved using:

curl "https://macaw.social/api/oembed?url=https://macaw.social/@andypiper/109349688112355529"
