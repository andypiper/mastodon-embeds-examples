This is a simple project to try out embedding content from the Fediverse (in this case, currently only Mastodon) using different tools and methods.

Toots are straightforward to embed (Mastodon's API supports oEmbed, and has an embed code option on the web UI), timelines need a bit more work.
  • Using Mastofeed - example
  • Using mastodon-embed-feed-timeline - example
  • A single Toot using the default option on Mastodon - example
  • A single unformatted Toot pulled from the Mastodon API - example
Also, there's a Fediverse sharing button on some of the pages.
 
Learn more and read the code by boopsing on the fishies!
